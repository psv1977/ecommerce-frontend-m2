# ⚓ YateLux - E-commerce de Yates de Lujo

Proyecto desarrollado para el **Módulo 2 (Frontend Estático)** del Bootcamp de Desarrollo Full Stack. Consiste en una plataforma de catálogo para la venta y reserva de embarcaciones exclusivas.

## 🚀 Requisitos Cumplidos (Rúbrica)

Este proyecto cumple con los siguientes criterios de evaluación:

* **HTML5 Semántico**: Uso estricto de etiquetas `<header>`, `<nav>`, `<main>`, `<section>`, `<article>` y `<footer>` para mejorar el SEO y la accesibilidad.
* **Diseño Responsivo (Bootstrap 5)**: Interfaz adaptada para dispositivos móviles (≤ 420px) y escritorio utilizando el sistema de grillas (Grid System).
* **Lógica en JavaScript**:
    * Renderizado dinámico de productos desde un arreglo de objetos.
    * Uso de funciones para la gestión del carrito de compras.
    * Persistencia de datos simulada mediante **LocalStorage**.
* **Control de Versiones**: Gestión de proyecto mediante Git/GitHub con flujo de ramas (`main` y `desarrollo`) y commits descriptivos.

## 🛠️ Tecnologías Utilizadas

* **HTML5 / CSS3** (Custom Styles)
* **Bootstrap 5.3** (Framework UI)
* **JavaScript (ES6+)**
* **Google Fonts & FontAwesome**
